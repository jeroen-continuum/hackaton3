Hieronder vindst je het overzicht van de geselecteerde bronnen, hun beleid (*Terms of Service*), de haalbaarheid van
scraping en de beschikbaarheid van API's.

Dit overzicht is zo opgebouwd dat de hackathon-teams direct weten welke bronnen ze via code kunnen aanspreken en welke
ze beter handmatig of via een omweg (zoals kant-en-klare open data downloads) kunnen gebruiken.

| Bron                                             | Beleid / ToS Status                                                                                          | Mag je scrapem?                                                                                                                       | API Beschikbaar?                                                                                                                         | Hackathon Aanbeveling & Alternatief                                                                                                                                                                              |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **KBO Open Data / Public Search**                | **Open Data:** Volledig hergebruik toegestaan voor commerciële en niet-commerciële doelen.                   | **Niet nodig:** Website scrapen is verboden, maar de overheid biedt de *volledige database* wekelijks gratis aan als download.        | **Ja:** Er is een officiële KBO-webservice, maar registratie/machtigingen duren te lang voor een hackathon.                              | **Beste keuze:** Download de volledige gratis CSV/XML-dump via de website van de FOD Economie en laad deze in een lokale database.                                                                               |
| **NBB Balanscentrale**                           | Openbaar archief, maar systematische massale extractie via de website wordt actief tegengegaan.              | **Nee/Moeilijk:** De website maakt gebruik van rate-limiting en ReCAPTCHA's op PDF-downloads.                                         | **Ja:** De NBB biedt een openbare API (via *Standardized Data* / XBRL-formaat) voor gestructureerde data van jaarrekeningen.             | **Aanbeveling:** Gebruik de officiële NBB Open Data API's om cijfers (zoals EBITDA en personeelsaantallen) direct op te halen via het ondernemingsnummer.                                                        |
| **Staatsbladmonitor.be**                         | Privé-initiatief dat data van het Belgisch Staatsblad aggregeert. ToS verbiedt commerciële scraping.         | **Nee:** ToS verbiedt geautomatiseerde extractie. Kans op IP-block.                                                                   | **Nee:** Geen gratis of publieke API beschikbaar.                                                                                        | **Alternatief:** Haal de publicaties rechtstreeks (en legaal) op via de officiële, gratis index van de **FOD Justitie (Belgisch Staatsblad)**.                                                                   |
| **LinkedIn (Free Tier)**                         | Extreem strikt anti-scraping beleid. Gebruik van bots schendt de ToS en leidt tot een permanente accountban. | **Absoluut Nee:** LinkedIn heeft zware technische beveiliging (Cloudflare, login-walls) en handhaaft dit juridisch en technisch hard. | **Beperkt:** Er zijn officiële API's, maar die geven geen toegang tot prospectiedata tenzij je een dure *Sales Navigator* licentie hebt. | **Alternatief:** Gebruik voor de hackathon een proxy-tool zoals de *PhantomBuster API* of *Proxycurl API* (commerciële scrapers met gratis proeftegoeden) om data legaal via gecertificeerde API's te verrijken. |
| **Jobboards (Jobat, Indeed)**                    | Commerciële platformen. ToS verbiedt scraping van vacatures voor externe databases.                          | **Nee:** Wordt actief geblokkeerd door anti-bot software.                                                                             | **Nee:** Alleen beschikbaar voor betalende enterprise partners of ATS-systemen.                                                          | **Alternatief:** Sla deze over of focus op de VDAB (zie hieronder).                                                                                                                                              |
| **VDAB (Jobboard)**                              | Publieke instelling. Ondersteunt innovatie en hergebruik van arbeidsmarktdata.                               | **Grijze zone:** Handmatig/beperkt mag, maar agressief scrapen wordt geblokkeerd.                                                     | **Ja:** VDAB heeft een uitstekend **Developer Portal** met gratis API's voor vacatures (na snelle gratis registratie).                   | **Aanbeveling:** Ideaal voor de hackathon. Registreer op het VDAB-ontwikkelaarsportaal om actieve vacatures van Belgische bedrijven live in te laden als indicator voor *Buyer Intent*.                          |
| **Sectororganisaties (Agoria, Assuralia, Voka)** | Openbare ledenlijsten bedoeld voor netwerken en visibiliteit.                                                | **Technisch Ja:** Hebben vaak nauwelijks bot-beveiliging. Check wel hun disclaimer voor commercieel hergebruik.                       | **Nee:** Geen openbare API's beschikbaar.                                                                                                | **Aanbeveling:** Omdat deze lijsten vaak klein zijn (enkele honderden bedrijven), is een eenvoudige Python-script (BeautifulSoup) op de online ledenzoeker de snelste manier om een startlijst te maken.         |
| **FOD Werkgelegenheid (Sociale Secretariaten)**  | Officiële overheidslijst van erkende instanties.                                                             | **Ja:** Het is een statische PDF/webpagina van de overheid met een dertigtal namen.                                                   | **Nee:** Geen API.                                                                                                                       | **Aanbeveling:** Dit zijn zo weinig daten dat een teamlid dit tijdens de hackathon in 2 minuten handmatig in een Excel-lijst kan plakken.                                                                        |
